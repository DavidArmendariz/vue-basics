# Vue intro

In these 3 small apps, you will learn about the basic of Vue:

- The declarative approach of Vue
- Data binding
- Event binding
- Listening for events with `v-on` or `@`
- Computed properties
- Watchers
- Styling
- Dynamic styling with object-based or array-based syntaxes
- `v-if`
- `v-else-if`
- `v-else`
- `v-show`
- `v-for`
- keys
